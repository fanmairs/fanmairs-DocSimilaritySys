    # 词向量模型说明

传统模式的同义词穿透功能支持加载 `.vec` 文本词向量文件（如 FastText）。

默认路径：

- `dicts/embeddings/fasttext_zh.vec`

如果该文件不存在，系统会自动退化为“同义词词典增强模式”（不会报错，只是语义穿透能力较弱）。

推荐做法：

1. 准备中文词向量 `.vec` 文件（可用 FastText 预训练中文向量）。
2. 放到本目录并命名为 `fasttext_zh.vec`，或在命令行通过 `--semantic_embeddings` 指定路径。
3. 启动传统模式时，系统会按当前语料词表按需加载，减少内存占用。
